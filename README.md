# 💊 Gambit Manager

Android installer and patcher application designed to install Gambit Xposed enhancements into Chess.com using LSPatch.

## 💪 Features

- 📲 **Install**: Patch and inject the Gambit Xposed module into Chess.com.
- 🔁 **Update**: Update Gambit to the latest version seamlessly.
- 📂 **Cached Base Support**: Load split/monolithic APKs directly from your local cache folder.

## 🔽 Download

Get the latest APK from the [releases page](https://github.com/Project-Gambit/gambit-manager/releases/latest).

## 🛠️ Building

To build the Gambit Manager APK:
```bash
./gradlew assembleDebug
```
